## Installation
Download [Python](https://python.org) and run `main.py`.

Virtual environment set up:
```
cd passive-aggressive-bot
python -m venv venv
venv/Scripts/python.exe -m pip install -r requirements.txt
```

Running the program:
```
venv/Scripts/python.exe main.py
```
